# Harness Skill Analysis 项目配置

## 项目目标
深入分析四个主流 AI Coding Agent 的 Harness Skill 项目（OpenSpec、Superpowers、GSD、Matt Pocock Skills），
提取设计模式、对比优劣、沉淀可复用的 Harness Skill 设计原则。

## 工作约定
- 使用中文撰写分析文档
- 分析时优先阅读源码，而非仅依赖 README
- 每个项目的分析结果放在 `analysis/<project>/` 目录下
- 跨项目对比内容放在 `docs/` 目录下

## 分析工作流
1. 阅读项目 README 和核心文档
2. 深入分析 Skill 源码结构和实现
3. 撰写结构化分析报告
4. 跨项目对比，提取通用模式和差异化设计

## 源码位置
- OpenSpec: `repos/openspec/`
- Superpowers: `repos/superpowers/`
- GSD: `repos/gsd/`
- Matt Pocock: `repos/mattpocock/`
